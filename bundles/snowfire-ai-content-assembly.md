## 3. Core Concepts (from Official Site & Blog)

### What Snowfire AI Is
Snowfire AI is an **Enterprise Decision Intelligence Platform** — not a content assembly tool. It:
- Aggregates internal and external signals into a single decision core
- Provides role-based decision visibility for each C-suite role
- Turns raw data into ranked, actionable recommendations
- Replaces static BI dashboards with adaptive intelligence

### Core Modules

#### Decision Collaboration
- Collaborative decision-making workflows
- Annotate and discuss data points in context
- Decision audit trails and version history

#### Signals Intelligence
- **Internal signals**: CRM data, ERP data, financial systems, HR data
- **External signals**: Market data, competitor intelligence, news, regulatory changes
- Signal processing and prioritization engine
- Real-time alerting on critical changes

#### Ask Your Data
- Natural language query interface
- AI-powered data exploration
- No-SQL querying for executives
- Context-aware answers

#### Metrics
- KPI dashboards per role
- Leading and lagging indicator tracking
- Goal progress visualization
- Variance alerts and root cause analysis

#### Connectors
- Pre-built connectors to enterprise systems
- Data source integrations (list from site)
- API-based data ingestion

#### Security & Governance
- Role-based access control (RBAC)
- Data residency and compliance
- Audit logging
- SSO/SAML integration

### Role-Based Views (16 C-suite roles)
Each role gets a tailored view showing only the metrics, signals, and recommendations relevant to their decisions:
- **CEO**: Company-wide health, strategic initiatives, risk overview
- **CFO**: Financial performance, cash flow, budgeting variance
- **CRO**: Revenue pipeline, churn signals, growth opportunities
- **CMO**: Campaign performance, market share, brand sentiment
- **CTO/CIO**: Tech stack health, deployment metrics, security posture

### Content Assembly (Legacy)
Snowfire's earlier positioning (content assembly / brand-compliant content) has evolved into the broader decision intelligence platform. Content creation is now one capability within the platform's output layer — generating formatted reports, board decks, and executive summaries.

## 4. API Reference

Snowfire AI's API capabilities include:
- **Connectors API**: Ingest data from enterprise systems
- **Query API**: Ask Your Data programmatic interface
- **Export API**: Export dashboards, reports, and decision artifacts
- **Signals API**: Subscribe to signal events programmatically

Authentication: Likely API key or OAuth 2.0
Integration: Pre-built connectors for common enterprise systems

For full API documentation: Contact Snowfire support or check in-app developer portal.

## 5. Integration Patterns

### Enterprise Data Connectors
- **CRM**: Salesforce, HubSpot
- **ERP**: SAP, Oracle, NetSuite
- **HR**: Workday, BambooHR
- **Finance**: QuickBooks, Xero, Expensify
- **Analytics**: Snowflake, BigQuery, Tableau, Looker
- **Communication**: Slack, Teams, email
- **External**: News APIs, market data providers, regulatory feeds

### Multi-Role Decision Workflow
1. **Signal Ingestion**: Data from 10+ sources flows into Snowfire
2. **Signal Processing**: AI prioritizes signals by urgency and relevance per role
3. **Role Routing**: Each executive gets their tailored view
4. **Recommendation Engine**: AI generates ranked action recommendations
5. **Decision Collaboration**: Team discusses and decides in-platform
6. **Action Tracking**: Decisions tracked to completion

### Board Reporting Pattern
1. Aggregated signals → executive summary
2. Role-specific deep dives
3. AI-recommended strategic actions
4. Export to PDF/presentation format

## 6. Agent Integration (MCP, Hermes)

### For AI Agents
Snowfire AI has a dedicated "For AI systems" support page, indicating intentional AI agent integration support:
- **MCP potential**: Snowfire could expose signals and decision recommendations via MCP
- **Query interface**: "Ask Your Data" is a natural language interface designed for AI consumption
- **Agent-to-agent**: AI agents can query Snowfire for decision intelligence data

### Suggested AI Agent Workflows
1. **Executive Summary Agent**: Agent queries Snowfire → generates daily briefing
2. **Decision Support Agent**: Agent monitors Snowfire signals → alerts on critical changes
3. **Report Generation Agent**: Agent pulls Snowfire data → creates board report
4. **Strategy Agent**: Agent analyzes Snowfire recommendations → suggests strategic pivots

### Hermes Integration
- Query Snowfire for executive intelligence
- Monitor critical signals via webhook/API
- Orchestrate multi-tool decision workflows
